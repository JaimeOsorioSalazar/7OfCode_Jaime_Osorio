7Days of Code

7Days of Code es un reto de 7 días el cual busca mejorar y afianzar conocimientos del lenguaje javascript. Cada día se estara enviando por correo electronico los requisitos para cada reto.

1/7 - Operadores logicos

Los operadores logicos buscan comparar dos o mas elementos con diferentes condiciones para poder realizar un resultado. Ahora bien aunque el numero 10 y "10" tengan el mismo valor, no tienen el mismo tipo.

Debido a esto, usar ciertos operadores dan resultados distintos.

2/7 - Interacción con Usuarios

Para el ejercicio del segundo día se debe crear una forma de aplicación que permita al usuario interactuar con la apliación lo que brinda una mayor experiencia al momento de interactuar con diferentes plataformas.

Nota: Al crear la aplicación que inteactue con el usuario se debe usar diferentes elementos tanto para el HTML como para el javascript

Respuesta

Para este ejercicio se usa principalmente "prompt" con el cual se abre una ventaja en el navegador y el usuario puede escribir alguna respuesta la cual puede ser almacenada para ser usada después. Usando "alert" se pueden brindar estas respuestas.

La pregrunta es, ¿Que otra forma demas de "prompt" puedo usar?

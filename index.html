Eliminar un elemento del array

Eliminar un elemento dentro de un array es algo complejo, ya que tenemos metodos como array.pop() metodo que elimina el ultimo elemento del array. El metodo array.shift() elimina el primer elemento del array. Pero existe una forma en la que puedes eliminar un elemento que no esta ni de primero ni de ultimo, array.splice() determinando la posición del alemento con array.indexOf() conseguimos la posicion dentro del array.

El array.indexOf() elimina elementos de cualquier posición pero no cuando el index es 0 ya que no elimina nada. En ese caso, usamos un if que determina si el index = 0 entonces ejecute array.shift() sino, entonces ejecute el array.splice().
